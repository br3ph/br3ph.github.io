What's this?

Here i will create an chatting Bot.

Where to use?

It is developed as a plug-in to @vysheng's telegram-cli. Since my machine is in a telegram group, i want it to join the conversation. So i decided to create a read-and-learn script. The script looks which questions we ask and which answers may fit and stores them.

How does it learn?

The users have to decide wheter an answer was good or not. With simple commands this is possible. answer-correct and answer-false will increase/decrease a fitting number in the database. If an answer reaches the fitting number zero, it will forget it. Some question might depends on the time of day, so it will store the time of the questions too. With this information the script is even able to ask if you slept well :)
